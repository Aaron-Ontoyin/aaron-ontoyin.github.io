GPTree Invention

GPTree is a general-purpose artificial intelligence algorithm designed to enhance decision-making through explainable, human-in-the-loop collaboration, eliminating the need for extensive feature engineering. When applied to venture capital (VC) investment analysis, GPTree significantly outperformed random selection by approximately 10x and expert benchmarks by about 3x. Fine-tuned for identifying outlier startups, it achieved an 18% success rate, underscoring its potential to transform early-stage investment strategies and drive superior fund performance.

Customer Churn Prediction

Designed and implemented a customer churn prediction system utilizing IBM’s telco customer dataset. Leveraged machine learning models including logistic regression, random forest, SVM, KNN, Naive Bayes, and XGBoost. Deployed the top three models via Streamlit, enabling users to prioritize recall, precision, or a balanced approach within the interactive application.

Advanced ML Pipeline for Network Intrusion Detection

Developed a robust machine learning pipeline for detecting network intrusions. Utilized Random Forest for feature extraction and PCA for dimensionality reduction, and implemented an optimized SVM classifier. Integrated the workflow using scikit-learn’s Pipeline object, with comprehensive performance evaluation using metrics such as accuracy, precision, recall, F1 score, ROC curve, and AUC-ROC. Enabled probability estimation and analyzed the impact of decision thresholds on detection performance.

OSAD API (Object, Speech and Audio Detection)

Engineered a test-driven, production-ready API for object detection in images and speech-to-text/text-to-speech conversion. The API features secure user authentication, session management, and password reset via email. Built with Flask, it employs TorchVision’s Faster R-CNN model for object detection, and integrates SpeechRecognition and pyttsx for audio processing.
